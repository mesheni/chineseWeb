# Китайское языковое приложение - План разработки

## Обзор
Веб-приложение для изучения китайского языка с серверным размещением. Минимальный прототип должен позволять пользователям учить ироганы, базовую лексику и грамматику.

## Технологический стек

### Backend
- **Node.js** + **Express.js** - сервер и API
- **SQLite** - база данных
- **Sequelize** - ORM

### Frontend
- **Vanilla JS/HTML/CSS** - минимум зависимостей

### Развертывание
- **Docker** - контейнеризация
- **docker-compose** - оркестрация

## Архитектура

## Функции (MVP)

1. **Изучение иероглифов**
   - Карточки с ироганами (SRS)
   - Отображение: ироган, пиньин, перевод, пример

2. **Лексика**
   - Базовые слова (500-1000)
   - Транскрипция и значения

3. **Грамматика**
   - 20-30 базовых правил
   - Простые упражнения

4. **Прогресс**
   - Счетчик изученных слов
   - Статистика

## API endpoints (MVP):
- `GET /api/words` - список слов
- `GET /api/words/:id` - детали слова
- `GET /api/random` - случайное слово

## База данных

Таблица `words`:
- id, character, pinyin, translation, example, category, difficulty

## Критерии успеха MVP
1. Сервер запускается без ошибок
2. API возвращает данные о словах
3. В браузере отображается интерфейс
4. Можно переключаться между карточками
5. Простая статистика сохраняется
